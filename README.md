# Typesnake
Snake game in typescript

#### Keys:

- W, Up Arrow (Turn up)
- A, left arrow (Turn Left)
- D, right arrow (Turn right)
- S, Down arrow (Turn down)
- R (restart)
- P (pause)
- C (continue)
